Features

Verge has many built-in features with many more to come this year. Heres a quick run-down of the main features
and what they do.


Tor and i2p

Tor (The Onion Router) is a service that allows you to hide your identity by moving your traffic between many Tor
servers and encrypting your data. It cannot be traced back to you as they would see the data originating from
one of the Tor servers

SPV

SPV stands for "Simple Payment Verification" which is technology that makes transactions take just 5-10 seconds.
It does this by instead of downloading a whole block, it verifies if a transaction is in a block and only downloads
block headers which are much smaller than full blocks.

Wraith Protocol

Wraith Protocol allows for fast, secure transactions by using technology called stealth addresses. This allows
desired transactions go to seperate stealth addresses which sends the transaction to the reciever. This transaction
is hidden on the ledger and impossible to trace.

Atomic Swaps

Atomic Swaps allows Verge to be used in DEX's (Decentralised Exchanges) which allows you to trade between Verge,
and other coins that have atomic swaps enabled, on the blockchain. This means that a company like Bittrex or CoinBase
dont own your coins but are instead stored in a decentralised exchange like crypto-bridge.

Mobile Security

Verge is the first and only cryptocurrency to have an Android wallet with Tor integrated. This allows for IP
masking and anonymity on the move. An i2p Androd wallet is expected in Q1 2018.

RSK Contracts

RSK contracts allows Verge to utilise smart contracts on the RSK blockchain. This will be enabled Q1 this year
and allows Verge to handle 1000 transactions per minute which is 10 times faster than what it is now. Below
is an image explaining what a Smart Contract is.

FAQ's

Q: Is Verge a Dogecoin fork?

A: No. While Verge did used to be called Dogecoin Dark, it was actually forked from Bitcoin/Peercoin

Q: Is Verge a scam?

A: No. Verge has been around since 2014, wasn't and ICO, and never had a premine. All assets are open source and avaliable on GitHub.
